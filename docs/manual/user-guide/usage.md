# InfoDive.exe 使い方ガイド

インフォグラフィック画像を題材に、パン／ズームのシナリオを組み立ててプレゼンテーションとして再生するWindowsデスクトップアプリです。

---

## 1. 起動

`InfoDive.exe` をダブルクリックで起動します。コマンドラインからの起動にも対応しています（後述）。

インストーラー（MSI）でインストールした場合は、スタートメニュー／デスクトップのショートカットに加え、`.przip` ファイルが InfoDive に関連付けられます。`.przip` をダブルクリックするとそのプロジェクトが直接開きます（起動時モードは `.przip` 内の設定に従う／Shift 押下でエディタ強制）。

また、オフライン動作を保証するため、インストール時に InfoDive の送信（インターネット）接続をブロックする Windows ファイアウォール規則が登録されます（アンインストールで自動削除）。

---

## 2. 基本の流れ（4ステップ）

1. **画像を開く** — ウィンドウ中央の円形ドロップゾーンへ画像をドラッグ＆ドロップ、またはクリックして選択
   - 対応形式: JPG / PNG / BMP / GIF / WebP / SVG / PDF / draw.io（`.drawio` / `.dio`）
   - PDF は各ページを個別画像として展開（`<元のファイル名>_p01.png` 形式、200dpi）
   - draw.io も全ページを個別画像として展開（詳細は [3.5 draw.io ファイルの取り込み](#35-drawio-ファイルの取り込み)）
   - 推奨解像度: 8192×8192px 以下（超過時は警告ダイアログ）
2. **ビューを調整** — キャンバス上でドラッグしてパン、マウスホイールでズーム
3. **シーンを登録** — サイドバーの「＋ 現在のビューをシーンに追加」でキーフレーム化
4. **プレゼン開始** — 右上の「▶ プレゼン開始」または `Ctrl+Enter`

---

## 3. 画像管理

- **画像を追加**: ツールバーの「画像を追加」— 既存画像を残したまま末尾に追加
- **画像差し替え**: 「画像差し替え」— 現在選択中の画像のみ別ファイルに置換（紐付くシーンは保持）
- **並び替え**: サイドバーの画像リストをドラッグ＆ドロップ
- **削除**: 画像リスト下のゴミ箱ボタン
- 複数画像対応: `Ctrl+Tab` / `Ctrl+Shift+Tab` で次/前の画像へ切替

---

## 3.5 draw.io ファイルの取り込み

draw.io / diagrams.net のファイル（`.drawio` / `.dio`）をドラッグ＆ドロップ、または「画像を追加」から取り込めます。複数ページの図は **全ページが個別画像** として展開されます（`<元のファイル名>_p01.png` 形式）。

取り込み方法はファイルの中身に応じて自動で切り替わります。

| ファイル種別 | 取り込み方法 | 必要なもの |
|---|---|---|
| **編集可能な SVG / PNG**<br>（draw.io の「Editable」エクスポート） | 内部で SVG/PNG として処理 | 不要 |
| **ネイティブ形式**<br>（通常の「保存」: mxGraph XML） | draw.io Desktop の CLI で PDF 化してから取り込み | **draw.io Desktop のインストール** |

### draw.io Desktop が必要なケース

通常の `.drawio`（mxGraph XML）を描画するには draw.io のレンダリングエンジンが必要なため、[draw.io Desktop](https://github.com/jgraph/drawio-desktop/releases) のインストールが必要です。以下の場所を自動検出します。

- `%LOCALAPPDATA%\Programs\draw.io\draw.io.exe`
- `%ProgramFiles%\draw.io\draw.io.exe`
- 環境変数 `DRAWIO_PATH` に実行ファイルのフルパスを指定（上記以外の場所にある場合）

draw.io Desktop が見つからない場合は、案内ダイアログを表示します。その際は draw.io 側で **PNG / SVG / PDF にエクスポート** してから取り込めば、追加インストール不要です。

> **注意:** draw.io Desktop が既に起動していると、変換結果が空になることがあります（draw.io の単一インスタンス制御のため）。その場合は draw.io をいったん終了してから再取り込みしてください。

## 4. シーン（キーフレーム）編集

サイドバーの各シーンカードから操作します。

- **▶ / ▼ チェブロン**: 詳細パネルの開閉
- **↑ / ↓**: 並び順を変更（ドラッグ＆ドロップでも可）
- **🗑 削除** / **📋 複製** / **✏ 名前編集**
- **↻ 現在ビューで上書き**: 開いているビューでシーンを更新
- **👁 プレビュー**: シーンへアニメーション遷移
- **遷移時間スライダー**: 0.2秒 〜 4.0秒
- **画像切替効果**: 無し / フェード / クロスフェード
- **絞り込み検索**: 🔍 検索欄でシーン名フィルタ

---

## 5. プレゼンテーション

### 表示スケールモード
ツールバーの「プレゼン表示」で選択:
- **横幅に合わせる** / **縦幅に合わせる**

### 起動時モード（.przip に保存）
ツールバーの「開いた時」で選択:
- **エディタ** / **プレゼン**

### プレゼン中の操作

| キー | 動作 |
|---|---|
| → / Space / PageDown | 次のシーン |
| ← / PageUp | 前のシーン |
| Shift+→ / Shift+Space | 次の画像 |
| Shift+← | 前の画像 |
| Home / End | 最初 / 最後のシーン |
| 1〜9 | シーン番号でジャンプ |
| Shift+1〜9 | 画像番号でジャンプ |
| F5 | 最初から再生 |
| B | ブラックアウト（任意キーで解除） |
| D | 描画モード ON/OFF |
| C | この画像の描画をクリア |
| E | 消しゴムモード |
| Ctrl+S | 描画をPNG画像として保存 |
| T | ツールバー表示/非表示 |
| H / ? | キーボードショートカット一覧 |
| Esc | 描画OFF → ガイド閉 → プレゼン終了（段階的） |

### 描画（アノテーション）
`D` で描画モードに入り、右上ツールバーで色（赤/黄/緑/青/白）と太さを選択。描画データは `.przip` には保存されません。残したい場合は `Ctrl+S` またはツールバーの「S」ボタンで PNG として書き出してください。

---

## 6. ファイル操作（.przip 形式）

プロジェクトは独自 `.przip` 形式（画像＋シナリオを ZIP 圧縮）で保存します。

| ショートカット | 動作 |
|---|---|
| Ctrl+N | 新規作成 |
| Ctrl+O | ファイルを開く |
| Ctrl+S | 上書き保存 |
| Ctrl+Shift+S | 名前を付けて保存 |
| Ctrl+Z / Ctrl+Y | 元に戻す / やり直し |

最近使ったファイルは「ファイル」メニューの「最近使ったファイル」から開けます。

---

## 7. エクスポート

「ファイル」→「エクスポート」から:
- **全シーンを PNG 連番で書き出し** — 各シーンを画像ファイル化
- **描画を PNG で保存** — 描画がある画像を PNG で書き出し（複数画像ならフォルダ選択）

---

## 8. ビュー操作ショートカット

| キー | 動作 |
|---|---|
| F | 画面にフィット |
| + / - | ズームイン / アウト（2%刻み） |
| Shift + ホイール | ズーム 10%刻み |
| F11 | 全画面切替 |
| Ctrl+\\ | サイドバー表示切替（メニュー経由） |

画像表示モード（「表示」メニュー）:
- 画面にフィット（デフォルト）
- 原寸表示（100%）

## 9. コマンドライン起動

```
InfoDive.exe [オプション] [ファイル ...]
```

### オプション
| オプション | 動作 |
|---|---|
| `-e` / `--editor` | エディタモードで開く |
| `-p` / `--present` | プレゼンモードで開く |
| (なし) | `.przip` に保存された起動時モードに従う |

### Shift キー起動オーバーライド
エクスプローラから `.przip` をダブルクリックする際、**Shift キーを押下したまま**起動すると `.przip` 内の保存済みプレゼンモード設定を無視してエディタモードで開きます。`--present` を明示指定した起動では従来通りプレゼンモードが優先されます。

### ファイル引数
- `.przip` を1つ指定 → プロジェクトを開く
- 画像 / PDF / draw.io ファイルを複数指定 → すべて読み込み（PDF・draw.io は 1 ページ＝1 画像として展開）
- 混在時 → `.przip` が優先され他は無視される

### 例
```
InfoDive.exe -e project.przip          エディタで開く
InfoDive.exe -p project.przip          プレゼンで開く
InfoDive.exe img1.png img2.png img3.png   複数画像を読み込み
InfoDive.exe report.pdf                PDFを全ページ展開して読み込み
InfoDive.exe diagram.drawio            draw.ioを全ページ展開して読み込み
```
